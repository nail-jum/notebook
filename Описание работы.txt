Тестовое задание выполнено на фреймворке Laravel 9.

Реализация API интерфейса
- маршрутизатор: routes/api.php
- контроллер: app/Http/Controllers/ApiController.php
- используемые типы запросов:
 get /notebook/ - получение списка записей
 post /notebook/ - добавление записи
 get /notebook/{id}/ - получение одной записи
 post /notebook/{id}/ - изменение записи
 post /notebook/delete/{id}/ - удаление записи

Также реализован веб интерфейс (не полностью)
- маршрутизатор: routes/web.php
- контроллер: app/Http/Controllers/NotesController.php

Для тестирования отдельных запросов использовалась программа Postman.
Для полноценного тестирования написана браузерная страница-имитатор API-приложения,
загружаемая по адресу:
/api-test

К сожалению, не все пункты задания удалось выполнить из-за отсутствия опыта и нехватки
времени. Но при необходимости могу выполнить задание в полном объеме.

Также, при необходимости, могу выполнить задание без использования Laravel, только на PHP.
